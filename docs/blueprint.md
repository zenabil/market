# **App Name**: Tlemcen Smart Supermarket

## Core Features:

- Product Catalog: Browse and search for products with detailed descriptions, images, and stock availability.
- Multi-Lingual Support: Support for Arabic (RTL), French, and English languages to cater to a diverse customer base.
- User Authentication: Secure user registration and login with email/password, Google, Facebook, and phone OTP.
- Smart Recommendations: AI-powered product recommendations based on user behavior and purchase history. The system is trained by purchase history. The genkit tool will then decide when it is appropriate to display similar items.
- Shopping Cart & Checkout: Manage items in a shopping cart, apply coupons, and proceed through a one-page checkout process.
- Order Management: Track order status and manage order history from the user account dashboard.
- Admin Dashboard: Manage products, inventory, orders, users, and content through a secure admin interface.

## Style Guidelines:

- Primary color: Warm sandy yellow (#F4E288) inspired by the Tlemcen landscape.
- Background color: Soft off-white (#F8F8F8) to create a clean and modern look.
- Accent color: Muted blue (#90AFC5) to provide subtle contrast and visual interest.
- Headline font: 'Belleza' sans-serif, for an artistic and design-centric feel.
- Body font: 'Alegreya' serif, to ensure readability and complement the header. This is particularly important for a trilingual interface.
- Use clear and consistent icons from ShadCN UI for product categories and actions.
- Clean and modern grid-based layout focused on product imagery.
